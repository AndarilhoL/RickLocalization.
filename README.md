# RickLocalization

A Cidadela do Rick's solicitou o desenvolvimento do RickLocalization.
O sistema deve manter os cadastros de Rick's e Morty's em todas as dimensões, dando acesso a uma ou mais dimensões.

### Tecnologias

- [.Net Core 3.1](https://dotnet.microsoft.com/download/dotnet/5.0)
- [EF Core](https://github.com/dotnet/efcore)
- [Angular](https://angular.io/)
- [Angular material](https://material.angular.io/)

### Padrões e Escolhas
- [Code First]
- [Auto Mapper]
- [Repository Pattern]
- [Rest API]

### :computer: Rodando o Projeto

```bash
# Ir para a pasta do Backend
$ cd Backend
$ dotnet run

# Em outro terminal ir para a pasta do Frontend
$ cd Frontend
$ cd RickLocalizationApp
$ ng serve
```
